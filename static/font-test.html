字體安裝

Linux

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip。
  3. 按兩下HanaMinB.ttf此一文檔。
  4. 按「安裝字體」。

Mac OS X

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip。
  3. 按兩下HanaMinA.ttf此一文檔。
  4. 按「安裝字體」。
  5. 按兩下HanaMinB.ttf此一文檔。
  6. 按「安裝字體」。

Windows

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip並把字體HanaMinA.ttf及HanaMinB.ttf存到電腦的某一個資料夾。
  3. 按一下 [開始]，再按一下 [執行]。
  4. 輸入 %windir%\fonts，然後按一下 [確定]。
  5. 按一下 [檔案] 功能表上的 [安裝新字型]。
  6. 按兩下包含字體的資料夾。
  7. 選取HanaMinA 及 HanaMinB兩個字體。
  8. 按一下以選取 [將字型複製到字型資料夾] 核取方塊。
  9. 按一下 [確定]。

IOS

  1. 首先你得安装Anyfont(或者RightFont)，把需要的字体通过分享打开，点击字体后再点开“设置”，安装描述文件。
